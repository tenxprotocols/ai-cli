# ai

A small, fast CLI for talking to LLMs. Supports Anthropic, OpenAI, Gemini, OpenRouter, and any OpenAI-compatible endpoint.

## Install

```bash
go install github.com/tenxprotocols/ai-cli/cmd/ai@latest
```

## Quickstart

```bash
export ANTHROPIC_API_KEY=...
ai what is the phase of the moon
```

See [docs/](docs/) for configuration, profiles, and the full command reference.
